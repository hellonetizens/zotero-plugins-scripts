directory\n```bash\ncd /home/zotero-plugins/zotero-metadata-plugin\nnpm install\n```\n\n### Issue: \"zip: command not found\"\n**Solution:** XPI builder uses Python's zipfile module (no external zip needed)\n\n### Issue: Build timeout\n**Solution:** Run scripts individually instead of using master orchestrator\n\n### Issue: Git push fails\n**Solution:** Ensure git is configured and you have push permissions\n```bash\ngit config --global user.email \"your@email.com\"\ngit config --global user.name \"Your Name\"\n```\n\n---\n\n## 📊 Performance Metrics\n\n### Build Times\n-
